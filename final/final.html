Final Description

Triggers

On Hover, Off Hover, On Click

Rules

  1. On Hover
    1. Add to cart button:
      • Background color changes to #004236
      • Text color changes #FFFFFF
      • Border disappears
      • Add to cart text moves slightly to the left
      • A cart icon fades in, shrinks in size, and settles on the right of add to cart text
      • Cursor changes to pointer
    2. Dropdown menu items:
      • Background color changes to #D5CD61
      • Cursor changes to pointer
  2. Off Hover
    1. Add to cart button:
      • Background color changes to #FFFFFF
      • Text color changes #004236
      • A #004236 border appears around add to cart button
      • Add to cart text moves back to the center
      • Cart icon disappears
      • Cursor changes back to normal
    2. Dropdown menu items:
      • Background color changes to #FFFFFF
      • Cursor changes back to normal
  3. On Click
    1. Add to cart button: A menu fades in and drops down, revealing the numbers of items can be added to cart
    2. Dropdown menu items:
      • A copy of the product image is created; it shrinks in size and flies toward the cart icon
      • Cart icon updates the number of items currently in cart
      • The database updates the number of items currently in cart
    3. Anywhere outside add to cart button and dropdown menu: Dropdown menu disappears
