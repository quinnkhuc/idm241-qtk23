  • Triggers:
    On-Hover, Off-Hover, On-Click
  • Rules:
    • On-Hover:
      • Change Add to Bag button background color from #004236 to #D5CD61
      • Change Add to Bag button text color from #FFFFFF to #004236
    • Off-Hover:
      • Change Add to Bag button background color from #D6CD61 to #004236
      • Change Add to Bag button text color from #004236 to #FFFFFF
    • On-Click:
      • Product image shrinks from 100% to 0% in size
      • Product image moves towards the cart icon on the top right corner of the screen
      • Cart icon updates the number of products user currently has on their cart
      • Update user cart in the database
