Beta Description

Triggers

On Hover, Off Hover, On Click

Rules

  1. On Hover
    1. Add to bag button:
      • Background color changes to #004236
      • Text color changes #FFFFFF
      • Border disappears
      • Add to bag text moves slightly to the right
      • A down arrow fades in, shrinks in size, and settles on the right of Add to bag text
    2. Dropdown menu items: Background color changes to #D5CD61
  2. Off Hover
    1. Add to bag button:
      • Background color changes to #FFFFFF
      • Text color changes #004236
      • A #004236 border appears around Add to bag button
      • Add to bag text moves back to the center
      • Down arrow disappears
    2. Dropdown menu items: Background color changes to #FFFFFF
  3. On Click
    1. Add to bag button: A menu fades in and drops down, revealing the numbers of items can be added to bag
    2. Anywhere outside Add to bag button and dropdown menu: Dropdown menu disappear
